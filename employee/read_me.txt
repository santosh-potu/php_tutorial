

Please follow the instructions to install the application

1) Unzip the dile employee.zip
2) copy the folder to your WWW or other directory where you place web applications

3) create one database with the name santosh
4) import file called santosh.sql (located in employee directory )  using phpmyadmin or other myssql client
5) configure the values in config.php (located in employee directory )

thats it! type http://localhost/employee  in your browser
